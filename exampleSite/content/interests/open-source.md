Open source software has been a cornerstone of my professional and personal journey. Contributing to and maintaining open source projects provides an opportunity to give back to the community while sharpening technical skills.

# Why Open Source

Open source fosters collaboration, transparency, and innovation. It allows security researchers and developers to build on each other's work, creating tools that benefit the entire community.

# Areas of Interest

- **Security tooling** — Static analysis, vulnerability scanners, and SBOM generators
- **DevSecOps automation** — CI/CD pipeline security tools and integrations
- **Mobile security** — Android security frameworks and testing tools
- **Documentation** — Making complex security topics accessible

# Community Involvement

- Regular contributor to security-focused open source projects
- Participant in Hacktoberfest and similar community events
- Speaker at open source conferences about security in open source
