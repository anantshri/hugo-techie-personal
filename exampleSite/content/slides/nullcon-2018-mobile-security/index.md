This talk goes beyond OWASP Mobile Top 10 to explore advanced mobile application security testing techniques. We cover runtime manipulation, certificate pinning bypass, binary protections analysis, and inter-process communication vulnerabilities on both Android and iOS platforms.

Attendees learn practical techniques for finding vulnerabilities that automated scanners miss, including custom Frida scripts, Objection workflows, and manual reverse engineering approaches.
